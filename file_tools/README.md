# File tools

This folder contains tools for file and folder operations.

Scripts:
- bulk_rename.bat: Rename files in a folder using a prefix and suffix.
- find_duplicate_files.bat: Find duplicate files in a folder and its subfolders.
- remove_empty_folder.bat: Scan a directory and remove empty folders.
- folder_size_analyzer.bat: Analyze the size of a folder and its subfolders and convert values to readable units.
- MD5_hash_calculator.bat: Calculate the MD5 hash of a file.
- SHA256_hash_calculator.bat: Calculate the SHA256 hash of a file.
