
# File tools

<br>

> [!IMPORTANT]
> Every script from this folder require administrator privileges to run.

<br>

## bulk_rename.bat
### Bulk rename files from a given folder with prefix and suffix.

<br>

## find_duplicate_files.bat
### Find duplicate files from a given folder and it's subfolder (The script does not delete those files.)

<br>

## remove_empty_folder.bat
### Scan and remove empty folder from a given directory

> [!WARNING]
> The script also delete subfolders.

<br>

## folder_size_analyzer.bat
### Anylise the size of a folder and it's subfolder and convert them into a readable unity (KB,MB,GB)

<br>

## MD5_hash_calculator.bat
### Calculate the MD5 hash of a given file.

> [!NOTE]
> For more information about MD5 hash see [this link](https://en.wikipedia.org/wiki/MD5).

<br>

## SHA256_hash_calculator.bat
### Calculate the SHA256 hash of a given file. Usefull to prove that a file is legit and not modified.

> [!NOTE]
> For more information about MD5 hash see [this link](https://en.wikipedia.org/wiki/Secure_Hash_Algorithms).
