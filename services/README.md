# Services

## service_check.bat
#### Script that check the state of a service. (Running or stopped)

## start_service.bat
#### Script that start a specified service

> [!IMPORTANT]
> This script require administrator privileges to run.

## stop_service.bat
#### Script that stop a specified service

> [!WARNING]
> Stopping services can have unintended consequences. Use this script with caution.

> [!IMPORTANT]
> This script require administrator privileges to run.

## restart_service.bat
#### Script to restart a specified service

> [!IMPORTANT]
> This script require administrator privileges to run.

## list_services.bat
#### List all services and their details on this computer

## list_services_name.bat
#### List all services on this computer but only display their names

## list_services_running.bat
#### List all running services on this computer

## list_services_stopped.bat
#### List all stopped services on this computer

## disable_service.bat
#### Script that disable a specified service.

> [!CAUTION]
> Disabled services can have unintended consequences (Can break your system). Use this script with extreme caution.

> [!IMPORTANT]
> This script require administrator privileges to run.

## enable_service.bat
#### Script that enable a specified service.

> [!IMPORTANT]
> This script require administrator privileges to run.
