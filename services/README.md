# Services

This folder contains scripts for managing Windows services.

Scripts:
- service_check.bat: Check the state of a service (running or stopped).
- start_service.bat: Start a specified service.
- stop_service.bat: Stop a specified service.
- restart_service.bat: Restart a specified service.
- list_services.bat: List all services and their details.
- list_services_name.bat: List all service names.
- list_services_running.bat: List running services.
- list_services_stopped.bat: List stopped services.
- disable_service.bat: Disable a specified service.
- enable_service.bat: Enable a specified service.

Many service scripts require administrator privileges.
